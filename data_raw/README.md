# Raw Data

This directory contains raw experimental datasets generated during the study.

## Contents

- ELISpot raw spot counts (SFU/well)
- Flow cytometry raw FCS-derived outputs (processed in FlowJo/FCS Express)
- Cytotoxicity assay raw percentage lysis values
- CD8⁺ T-cell expansion raw counts

## Notes

All raw data correspond to experiments performed using PBMC-derived CD8⁺ T cells stimulated with HPV16 E6/E7-derived peptides.

Raw datasets were used as inputs for downstream statistical analysis and visualization in `/scripts_r`.

No statistical transformations or normalization have been applied at this stage.

## Usage

These files serve as the primary source for reproducibility of all figures and statistical analyses reported in the manuscript.
