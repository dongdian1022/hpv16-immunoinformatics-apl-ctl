# R Analysis Scripts

This directory contains R scripts used for statistical analysis and figure generation.

## Software Environment

- R version: 4.4.3
- Packages:
  - ggplot2 (v4.0.3)
  - dplyr (v1.2.1)

## Contents

- ELISpot analysis scripts
- CD8⁺ T-cell expansion analysis
- Cytotoxicity assay statistical tests
- Data visualization scripts for Figures 3–6

## Statistical Methods

- Student's t-test (paired or unpaired, as appropriate)
- Welch’s t-test for unequal variance comparisons
- One-sample t-test against control baseline (Mock-T)
- Data presented as mean ± SD or SEM as specified in the manuscript

## Output

Scripts generate:
- Publication-quality figures
- Summary statistics tables
- P-value annotations used in manuscript figures
